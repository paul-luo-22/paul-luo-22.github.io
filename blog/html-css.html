Class & ID

Classes and IDs are selectors we can use to choose elements on an HTML page to manipulate. While classes are not unique and can be used for multiple elements an ID selector can only be used for a single element that appears a single time on the HTML page. Given that both tags are the same if we were to apply a background color to the first paragraph we must have a way of differentiating between the two. We can give the first paragraph a class or id to differentiate the first from the second. Given that we are only changing one element we can use an ID.

Note that in the HTML we declare our id "first-paragraph" and in the CSS we reference it with #first-paragraph. Once again let's take a look at the output below.

Classes however are used when we want to apply that change to multiple elements which in this case is both paragraphs. Note that there is a change in syntax.

Here we use declare the class of both elements to be "example-paragraph" in the HTML and in the CSS we reference it with .example-paragraph instead of using #. Once again we can see the final output below.
